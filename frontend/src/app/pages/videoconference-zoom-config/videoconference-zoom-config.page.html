Sin prueba reciente

Usa el boton de prueba para confirmar que las credenciales guardadas ya permiten hablar con la API de Zoom.

1. Guarda antes de operar

La verificacion de licencias del pool y el sync con Zoom dependen de estas credenciales guardadas.

2. Mantener `maxConcurrent` global

Ese valor sigue aplicando a todos los hosts del pool por igual, tal como vienes manejando las licencias.

3. Si falla la prueba

Revisa `accountId`, `clientId`, `clientSecret` y que la app Zoom tenga permisos para listar usuarios y reuniones.
